c) Número de expediente: o indicado para cada expediente.

a) Descrición do obxecto: a indicada para cada expediente.

b) División por lotes e número: os indicados en cada expediente.

c) Lugar de entrega/execución: A Coruña ou Ferrol segundo o especificado en cada expediente.

d) Prazo de entrega/execución (meses): os arriba sinalados.

3. Tramitación, procedemento e forma de adxudicación.

a) Tramitación: urxente.

b) Procedemento: aberto.

c) Forma: concurso público.

4. Orzamento de liciación.

a) Importe total: os indicados para cada expediente.

b) Prima: a indicada para cada expediente.

c) Canon: o indicado para cada expediente.

5. Garantías.

a) Definitiva: a indicada para cada expediente.

6. Obtención de documentación e información.

a) Entidade: Universidade da Coruña. Servicio de Xestión Económica.

b) Enderezo: rúa Maestranza, s/n.

c) Localidade e código postal: 15001 A Coruña.

d) Teléfono: 981 16 70 00.

e) Telefax: 981 16 70 12.

f) Data límite de obtención de documentos e información: o día anterior ó de finalización do prazo de presentación de ofertas.

7. Presentación das ofertas.

a) Data límite de presentación: o décimo terceiro día natural seguinte ó da publicación desta resolución no DOG.

b) Documentación que hai que presentar: a indicada nos pregos de cláusulas administrativas particulares.

c) Lugar de presentación:

1. Entidade: Universidade da Coruña. Rectorado. Rexistro xeral.

2. Enderezo: rúa Maestranza, s/n.

3. Localidade e código postal: 15001 A Coruña.

d) Prazo durante o que o licitador estará obrigado a mante-la súa oferta: 3 meses.

e) Admisión de variantes: admítense ata tres variantes.

8. Apertura das ofertas.

a) Entidade: Universidade da Coruña.

b) Enderezo: rúa Maestranza, s/n.

c) Localidade: A Coruña.

d) Data: terceiro día seguinte ó do remate do prazo de presentación de proposicións: se este día fose inhábil ou sábado, realizarase o seguinte día hábil.

e) Hora: 11 horas.
